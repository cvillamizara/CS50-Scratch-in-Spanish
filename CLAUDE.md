# CS50 Scratch — Versión en Español

## Project Goal
Build a Spanish translation of https://cs50.harvard.edu/scratch/ for Spanish-speaking kids in underprivileged communities in Latin America who cannot access the English version.

## Audience
Children and young learners in Spanish-speaking regions (primarily South America). Many come from low-resource backgrounds. The site must feel welcoming, accessible, and structurally familiar to the original CS50 Scratch page so kids can eventually transition to the English version.

## Structure philosophy
Match the CS50 Scratch page layout as closely as possible. Kids who later visit the English site should feel at home.

## Tech stack
Plain HTML + CSS + JS. No build step. Modular: separate files for styles, scripts, and components. Easy to deploy on GitHub Pages or any static host.

## Tone
Warm and close like CS50's own voice — not informal (no "tú" slang), not overly formal. Think: a great teacher talking to students, not a textbook.

## Attribution (required on every page)
Este curso es una adaptación en español de [CS50's Introduction to Programming with Scratch](https://cs50.harvard.edu/scratch/) de Harvard University. Todo el contenido original pertenece al equipo de CS50. Distribuido bajo licencia Creative Commons.
